Links and Images

Create a link with an <a> tag

Use the href attribute to add a destination where we want our linke to go.

<a href="www.google.ca">Google</a>


While href specifices the desination link, targer specifies where the linked resource will be opened. If it is not present, then, by default, it will take on the _self value which opens the link in the current tab. To open the link in a new tab or window (depends on browser settings) you can set it to _blank as follows:

<a href="www.google.ca" target="_blank" rel="noopener noreferrer">Google</a>


Display an image in HTML using the <img> tag.

To link an image use the src attribute

<img src="https://www.theodinproject.com/mstile-310x310.png">

Whille no required, specifying the height and width attrtributes in image tags helps the browser layout the page

Use the Width and Height attributes

<img src="https://www.theodinproject.com/mstile-310x310.png" height="310" width="310">

However it is recommended you leave all styling for CSS so there is no confusion
